# 04-UI-BRIEF.md
## Storefront Design Context
Modern premium e-commerce look with Google Stitch design patterns:
- Strong whitespace with clean layout
- High-quality large product images
- Smooth product listing transition to detail page
- Clear, modern checkout funnel
- Admin should be simple and functional
- Include suggested Stitch design prompts but no external links

## Design Language
- Clean card layouts
- Subtle gradient backgrounds
- Product images with shadow and hover effects
- Product cards with fast, clear transitions
- Checkout should show order summary clearly on each step
- Admin should be minimal and readable

## Example Stitch Prompts
"Convert this page to Google Stitch design: hero image with gradient overlay and product carousel"
"Create modern store header with floating action cart badge"
"Make product card with shadow and hover zoom on image and price change"
"Checkout page with step progress and clear validation"
"Admin panel with dark card layout and product edit cards"

## Mobile-first Approach
- Responsive layout that adapts to all screen sizes
- Mobile checkout funnel with simple forms
- Admin should be readable on mobile
