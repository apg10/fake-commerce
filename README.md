# fake-commerce

Controlled full-stack fake e-commerce project used to test local AI coding workflows with OpenCode, Ollama, Qwen models, GPT-OSS, and Codex.

Current backend:
- FastAPI
- Pydantic v2
- Pytest
- In-memory product API
- In-memory category API

Current status:
- Product API supports create, list, retrieve, update, delete, filtering, and pagination.
- Category API supports create, list, retrieve, and partial update.
- Database integration is planned for a later phase.
